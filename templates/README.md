# Validated GKE Templates

This directory contains Infrastructure as Code (IaC) templates that have been automatically generated and validated in a sandbox environment.

## Structure
Each template is stored in a subdirectory named after the issue number and a short name:
`templates/{issue-number}-{short-name}/`

## Usage
Refer to the README.md within each template directory for specific deployment instructions.
